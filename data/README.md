# Dáta a Datasety

Tento adresár obsahuje dáta a datasety vytvorené v rámci bakalárskej práce.

## Štruktúra

- `raw/` - Surové, nespracované dáta
- `processed/` - Spracované a vyčistené dáta
- `annotations/` - Anotácie a labely
- `splits/` - Rozdelenie na trénovaciu/validačnú/testovaciu množinu

## Poznámky

Z dôvodu veľkosti súborov nie sú v tomto repozitári uložené všetky dáta. 
Kompletný dataset je dostupný na: [odkaz na dataset]

## Formát dát

Dataset je organizovaný v nasledujúcom formáte:

```
data/
├── images/           # Obrázky
├── labels/           # Anotácie
└── metadata.json     # Metadáta datasetu
```
